# Проект интернет-магазина (в разработке)

## Приложение catalog
Приложение содержит HTML-шаблоны для домашней страницы и для страницы с контактной информацией. Созданы контроллеры для отображения соответствующих страниц, настроена маршрутизация.
На странице контактов реализована форма обратной связи.
